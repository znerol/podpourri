podpourri-job-local
===================

Synopsis
--------

**podpourri-job-local** *job-tag-prefix* *repo-url*


Description
-----------

Runs a build job by executing ``podpourri-build`` locally inside a temporary
checked out repository. This is normally called by ``podpourri-job``.

--------
